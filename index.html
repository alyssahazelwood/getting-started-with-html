Intro to HTML

HTML = hypertext markup language
It is the basis for everything on the web
HTML is made up of: text content, markup, references to other docs,
and links to other pages

Markup is the part of the HTML that you don't see.
Content is what displays on a webpage
An element is everything within a set of HTML tags.

Tags

Tags communicate to the computer what type of element you are inputting
We learned the following tags:
break
bold
italics
paragraph and break
span and div
Image
Attribute and Anchor

If a tag that requires an end tag is missing one, the remainder of the text will follow the
instruction to the end of the text.
A void tag does not require an end tag.
Tags can be "inline" or "block"
A block tag creates a box around the content--needs an end tag. A box can have height and width.

Images and Attributes

You can insert images with the image tag or links with the attributes tag.
An attribute is the property of an HTML tag.
